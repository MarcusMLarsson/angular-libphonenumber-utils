Angular Phone Number Utilities

Seamlessly integrate Google's powerful libphonenumber library into your Angular app with these ready-to-use tools. Click the links below to explore their functionality and see the code behind them.

  • Phone Number Pipe: Format phone numbers according to the selected locale in various styles, such as international, national, E164, and RFC3966.
  • Phone Number Directive: Automatically format phone numbers as you type, applying locale-based rules and country code validation.
  • Phone Number Validator: Validate phone numbers based on the selected locale using Google's libphonenumber library.

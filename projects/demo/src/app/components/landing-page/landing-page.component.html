Welcome to Phone Number Utilities

Angular utilities for phone number formatting, validation, and input handling using Google's libphonenumber library.

This application provides an easy-to-use interface for handling phone numbers, including the following features:

  • Phone Number Validator: Validate phone numbers according to the selected region using Google's libphonenumber library.
  • Phone Number Pipe: Format phone numbers in different formats such as international, national, E164, and RFC3966.
  • Phone Number Input Directive: Automatically formats phone numbers as you type, applying country code validation.

Directive Demo

This demo showcases an Angular directive (`czPhoneNumberDirective`) that formats phone numbers dynamically as you type, based on the selected region. This is an example of how you can structure the directive. The `czPhoneNumberDirective` formats phone numbers dynamically in the view using the selected region. This is how you can use the `czPhoneNumberDirective` in the view. It works with an input field and binds to the `regionCode` to format the phone number.
